## 一、9 屏逐屏拆解（元素 + 关系）

### 屏 1 · 行情首页（`35301af4…`）
**元素**：个股搜索框（宁德时代）→ 沪深京/港股/美股 切换 → **开户引导进度条**（进入开户·填写资料·提交申请三段式）→ 三大指数卡（上证 4031.51 +1.12% / 深证 / 创业板，**红涨**）→ **「市场总览」温度计仪表盘：76°C「微热」+ 当日分时线** → 涨跌分布/成交额 切换 → AI 解读资讯流（"沪指涨 1.12%，资金高低切换"）→ **底部 AI 对话输入框**（"明日市场风格会延续高低切换吗?"）。
**关系**：温度计把"全市场情绪"压缩成一个**单一可读指标**（76°C/微热），是这屏的视觉锚；指数卡是事实，温度计是"翻译"，AI 资讯/对话是"解读"——事实→翻译→解读 三层递进。

### 屏 2 · 股票自选（`3d9d1b10…`）
**元素**：上证指数小条 + 基金自选 入口 → 沪深/港股/美股/全部 → **筛选 chips（今日主力净流入≥1亿 / 股价创30天新高）** → 自选股列表（宁德时代/微软/亚马逊/谷歌/苹果，每行 **mini sparkline + 价格 + 红绿涨跌 pill**）→ 自选动态（资讯/异动/资金 三 tab）→ 自选晚报（29 要闻/1 公告/0 研报）+ 自选大事 双卡。
**关系**：自选 = 用户的"关注集合"，列表给"价格事实"，自选动态/晚报给"为什么动"。涨跌 pill 是红绿强对比。

### 屏 3 · 基金持仓页（`9205ec88…`）⭐ 与本项目最相关
**元素**：蓝色主题 + "资金安全有保障" + 机构专享 → **总金额 3,004.24 + 👁 隐藏 toggle** → **三栏收益拆解：昨日收益 −10.39 / 持有收益 +404.24 / 累计收益 +74.38** → 我的总资产 → **金刚区：收益明细 / 持仓分析 / 交易记录 / 投资计划 / 清仓分析** → 市场热点轮播 → **我的持有 + 风险分类 tabs（全部/偏股/偏债/指数/黄金/全球）** → 持仓行（净值 + 金额 + 收益额/率）。
**关系**：这屏是支付宝理财页的精髓所在——**总资产（单焦点）→ 收益拆解（翻译涨跌为钱）→ 金刚区（高频动作下钻）→ 风险分类（教育性组织）**。`03` 的 C-1/C-2 正对应此屏。

### 屏 4 · 基金自选（`cc873970…`）
**元素**：每日必看/基金对比/提醒服务 → **「发现」推荐：均衡方向入选 4 月值得投 — 华安事件驱动量化策略混合C +17.05%（均衡方向/稳健投向 tag）** → 全部/持有/已清仓/偏股/偏债/指数 → 基金列表（净值 + 涨跌幅 + tag：持有/跑赢同类/跑赢大盘）。
**关系**：用"标签"（均衡/稳健/跑赢同类）给基金贴**风险与业绩双标签**，降低小白决策成本。

### 屏 5、6 · 更多·功能矩阵（`ca299480…` / `d2cfc53d…`）⭐ 游戏化金矿
**元素分区**：
- **首页服务**（可编辑图标行 + **编辑** 按钮 → 个性化金刚区）
- **行情工具**：财富号/热门股票/热门板块/打新日历/资金流向/ETF专区
- **选股诊股**：选股/热门ETF/**多维诊断**/龙虎掘金/两融选股/**机构预测**/**扫雷宝**/选股工具/**研报透视镜**/天天看牛股
- **活动权益**：**波动红包 / 模拟炒股 / 大盘晴雨 / 猜涨跌 / 体验金 / ETF大赛** ← 直接是游戏化机制
- **社区资讯**：今日收评/自选早晚报/市场情报/VIP资讯/陪你看盘/走进上市公司
**关系**：把几十个功能按"工具/诊断/活动/内容"四类收纳，**活动权益区专门承载游戏化激励**，与本项目的需求高度契合。

### 屏 7 · 基金市场·抓机会（`92fddc68…`）
**元素**：金选锂健 品牌升级 banner（"市场波动太大?来点稳的!"领首发礼）→ **金刚区：基金排行/热门板块/金选好基/值得投/潜力信号/大数据/偏股基金/偏债基金/指数+/全球投资/新发基金/红利** → 我的基金动态 → **精选/抓机会/投资圈 tabs** → 更多投资线索（综合/投资方向/基金类型）→ **机构共识度概览（半导体 34家 / 短债 34家 / 芯片半导体 33家 / 人工智能…）**。
**关系**："机构共识度"用"多少家机构看好"做**社会认同信号**，是低成本的决策辅助与教学锚点。

### 屏 8 · 股票发现页（`b7a3dc0b…`）
**元素**：**ETF股林大会·玩ETF大赛·瓜分百万奖池**（机器人插画 banner）→ 金刚区（财富号/波动红包/热门股票/热门板块/ETF专区/打新日历/热门ETF/模拟炒股/选股/更多）→ 双营销卡（瓜分百万奖池 GO / **学投资课程·领专属·最高30元消费红包**）→ A股市场（已收盘，成交额 32150 亿，三大指数）→ 打什么/今日风口（玻纤/英伟达）→ 资讯流。
**关系**：**赛事（ETF大赛）+ 学习激励（学投资领红包）** 并列在首屏黄金位——这正是"竞争力 + 学中学"双驱动的范例。

### 屏 9 · 机会·值得投（`fe4837f8…`）
**元素**：**6月值得投·连续30个月公布** badge → 长期机会 chips（低波固收+/科技/跑赢大盘/黄金/红利）→ 阶段热点（高端制造/新能源/煤炭）→ **低波固收+ 卡（政策宽松/配置良机/攻守兼备 + 具体债基）「投一笔」** → 科技卡（AI革命）→ 热门板块·实时解读（**黄金概念 8天首次资金净流入 +6.03%**）→ 涨幅领先/资金流入/搜索飙升/估值低 tabs。
**关系**："连续30个月公布"是**信任/一致性信号**；"投一笔"把建议直接转成一键动作（建议→行动 零摩擦）。

## 三、新增功能全栈说明（C-8 / C-9 / C-10）

> 沿用 `03` 的风险红线：所有游戏化激励**只给装饰性奖励**，绝不直接发战力，避免重开刷分路（守住 `01` 的 DOM-01/02/03 修复成果）。

### C-8 「市场温度计」情绪指标（对应屏 1）
- **教学目标**：把当回合全市场的涨跌、波动、事件压成一个 0–100°C 的"市场情绪温度"，教学生识别"过热（贪婪）/ 过冷（恐慌）"，引出逆向思维。
- **游戏化**：温度计随回合波动的拟物动效（克制，遵 FE-13 不滥用动画）；"极热/极冷"时给一句反人性提示（"别人贪婪我恐惧"），与现有 `adaptive-events` 的行为干预一脉相承。
- **UI**：放在 `/student/market` 顶部，温度计 + 一句情绪标签 + 当回合分时/净值微图。
- **前端**：`src/components/student/market-thermometer.tsx`（纯 SVG，`"use client"` 仅做指针动画）。
- **后端·逻辑**：纯函数 `computeMarketTemperature(run, round, eventTimeline)` 放 `src/lib/market-data.ts` 或新 `src/lib/market-sentiment.ts`，**由 `seed`/`eventTimeline` 决定**（确定性、可单测、可复现，符合 `determinism.guard.test.ts` 约束）。无新 API（随 sim-state 返回）。
- **数据库**：无新表。
- **验收**：同一 seed 同回合温度恒定；极值触发逆向提示；纯函数单测覆盖边界（全涨/全跌/无波动）。

### C-9 「活动权益中心」游戏化货架（对应屏 5/6/8 活动权益区）
- **教学目标**：把"猜涨跌""模拟小赛""波动红包"做成低门槛、高频的微练习，让学生在轻量博弈里建立"预测有不确定性"的直觉。
- **游戏化**：
  - **猜涨跌**：每回合推进前，学生猜下回合某指数/标的涨或跌；猜中得**装饰性积分/称号**（不发战力）。教学点：预测胜率回归 50%，亲身体会"择时难"。
  - **模拟小赛 / ETF大赛 → 复用现有战力榜/赛季榜**，作为活动入口聚合，不另起炉灶。
  - **体验金**：新学生赠一笔"体验金"做无压力首单（仅练习 run，不进公平榜）。
  - **波动红包**：完成"分散投资/坚持定投"等教学行为后开红包，奖励为装饰皮肤/头像框。
- **UI**：`/student/quests` 升级为"活动权益中心"，按"练习/赛事/福利"三类收纳（呼应屏 5 的四分区收纳法）。
- **前端**：复用 C-6 任务中心组件，新增"猜涨跌"卡（`"use client"` 提交猜测）。
- **后端·API**：`POST /api/sim/predict`（提交本回合猜测，`checkOrigin`+限流+防双击 FE-01，结算在回合推进时）；猜测结果与红包结算走幂等。
- **数据库**：`round_predictions(user_id, run_id, round, guess, resolved, correct)`；装饰奖励 `achievements`（与 C-6 共表）。
- **验收**：猜测在回合推进时结算且只结算一次；预测**不影响**净值与战力（纯装饰激励）；体验金 run 不进公平榜。

### C-10 「机构共识度 / 同学热度」社会认同卡（对应屏 7）
- **教学目标**：展示"多少人/多少同学本回合在买某标的"，教学生理解羊群效应——**展示 ≠ 推荐**，并配一句"热门不等于适合你"的反思提示。
- **游戏化**：班级内"本回合最热标的 Top3"榜（聚合，不暴露个人），制造话题与竞争感。
- **UI**：市场页一张"同学热度"卡（脱敏聚合）。
- **前端**：只读卡片。
- **后端·API**：`GET /api/market/peer-heat`（**按班级聚合**，复用 BE-01 修复后的班级过滤；只返回计数，绝不返回个人持仓——隐私红线）。
- **数据库**：从 `scenario_runs.holdings` 聚合，无新表；**必须脱敏**（仅 count，不含 userId）。
- **验收**：只暴露聚合计数、零个人信息泄露；冷启动（无数据）有空态；配反思文案不诱导跟风。
