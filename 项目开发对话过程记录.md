# Brown Zone / stu-company 项目开发对话过程记录

> 本文档用于归档本轮网页端应用从需求研读、功能开发、界面优化、部署上线到本地迁移方案的主要对话过程。为避免泄露敏感信息，文档不记录任何真实 API Key、访问令牌或私密环境变量值。

## 项目概览

- 项目名称：Brown Zone / 幕后之手
- 产品名称：Mr.Brown AI 经济沙盘
- 仓库拟命名：stu-company
- 技术栈：Next.js App Router、TypeScript、Tailwind CSS、Framer Motion、Drizzle ORM、Supabase/Postgres 预留、本地内存 store、Anthropic-compatible AI 接口封装、AllTick 行情接口封装。
- 核心定位：面向学生财商教育与经济学课堂的网页端模拟沙盘，包含官网展示、学生端、教师端、家长端、管理端、AI 导师、行情观察、历史复盘与本地课堂演示能力。

## 当前功能清单

- 官网首页：品牌 Hero、产品叙事、功能模块、股票轮播条、课程与试玩入口、响应式导航。
- 学生端：策略总览、市场信息、历史复盘、课程模块、模拟交易、储蓄/贷款/房产/创业、排行榜、AI 导师点评。
- 市场页：AI/科技观察池、AllTick 行情接入、10 分钟刷新、本地教学观察池 fallback、K 线、雷达、排行、行业热度。
- 历史复盘：回合趋势、历史操作分组、关键回合、AI 总结与下一步建议。
- 教师端：班级、邀请码、任务、学生表现与排行榜。
- 家长端：绑定学生、成长曲线、风险提醒与阶段报告。
- 管理端：基础内容管理与邀请码管理。
- KeyAI：全站悬浮 AI 助手，多轮对话、历史记录、本地兜底模式。
- 本地演示：一键 bat 启动、离线可用、教师电脑迁移说明。

## 安全与上传说明

- 不上传 `.env.local`，其中包含本地私密环境变量。
- 不上传 `node_modules`、`.next`、`.vercel`、日志文件和本地临时文件。
- `.env.example` 仅保留变量名和示例端点，不包含真实密钥。
- 远端 AI 与 AllTick 行情在无密钥或无网络时会自动降级为本地教学兜底。

## 本地运行摘要

```bash
npm install
npm run dev
```

也可以在 Windows 上双击：

```text
一键启动BrownZone.bat
```

默认访问：

```text
http://127.0.0.1:3000/demo
```
