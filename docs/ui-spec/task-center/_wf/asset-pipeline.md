## 1. New output directory layout under `public/mascots/`

The existing `quest-world/characters` art is baked-background and tuned for the small in-card avatar circle. The upgrade needs **transparent-subject** mascots that the frame renderer composites onto rarity frames. Put the new series under a fresh root so the old assets keep resolving during migration:

```
public/mascots/
  portraits/                 # transparent-subject 3D mascot, 1:1, the hero asset
    <slug>.webp              # e.g. steady-bear.webp  (~44KB, 768ВІ)
    <slug>@2x.webp           # retina, 1024ВІ        (~70KB, only for celebration hero)
  frames/                    # rarity frame plates (mostly SVG, art only where foil needed)
    common.svg
    rare.svg
    epic.webp                # epic/legendary need raster foil/glow вҶ’ WebP
    legendary.webp
  celebration/               # иҺ·еҫ—еј№зӘ— editorial poster backplates (solid duotone)
    <slug>-poster.webp       # per-mascot, matches В§4 reference image
  fallback/
    mascot-fallback.webp     # neutral claymation "?" toy on slate вҖ” never 404
    frame-fallback.svg
manifest.mascots.json        # generated by the pipeline; the UI reads slugвҶ’meta from here
```

Naming rule: **`slug` is kebab-case English** and is the single join key across `portraits/`, `celebration/`, the deck `card.id`, and `manifest.mascots.json`. Where a mascot maps to an existing card, **reuse that `card.id` as the slug** (e.g. `calm-observer`, `risk-shield`) so `QuestCardArt` can be upgraded to read `mascots/portraits/${card.id}.webp` with the old `quest-cards/front-${card.id}.png` as the `onError` fallback вҖ” zero breakage.

## 4. Card-frame treatments per rarity tier

Frames are the *foil/ribbon/glow chrome* composited around the mascot portrait. **Common and rare are pure SVG** (crisp, weightless, themeable via the mascot `hex`); **epic and legendary need raster foil/glow** so they ship as WebP. The frame is a separate layer from the mascot so any mascot can be re-skinned to any tier without regenerating art.

## 5. Generation + optimize pipeline (`scripts/gen-mascot-images.mjs`)

A near-copy of `scripts/gen-market-images.mjs` вҖ” same gateway (`https://gpt-agent.cc/v1/images/generations`), same `gpt-image-2` model, same `AI_API_KEY` env contract, same idempotent skip, same 3-worker `pool()`, same retry/size-fallback loop. Three deltas: (a) **transparent PNG** handling (`size:"1024x1024"`, request transparency, keep alpha through sharp), (b) **two outputs per portrait** (`<slug>.webp` 768ВІ + `<slug>@2x.webp` 1024ВІ for the celebration hero), (c) **manifest emit**.

### 5.1 The script (drop-in)

```js
// scripts/gen-mascot-images.mjs
// Generate the mascot IP series + epic/legendary frame foils for д»»еҠЎдёӯеҝғ.
// Mirrors scripts/gen-market-images.mjs (gpt-image-2 via gpt-agent.cc, sharpвҶ’WebP).
// Idempotent, concurrency-limited, retried. Emits public/manifest.mascots.json.
//   $env:AI_API_KEY="<key>"; node scripts/gen-mascot-images.mjs            (all)
//   node scripts/gen-mascot-images.mjs calm-observer compound-keeper       (subset)
import { writeFileSync, existsSync, mkdirSync, statSync } from "node:fs";
import { fileURLToPath } from "node:url";
import { dirname, join } from "node:path";
import sharp from "sharp";
import { MASCOTS, MASCOT_STYLE, RARITY_LOOK } from "./mascot-prompts.mjs"; // В§3 tables

const __dirname = dirname(fileURLToPath(import.meta.url));
const ROOT = join(__dirname, "..", "public", "mascots");
const PORTRAIT_DIR = join(ROOT, "portraits");
const KEY = process.env.AI_API_KEY;
if (!KEY) { console.error("зјәе°‘ AI_API_KEY зҺҜеўғеҸҳйҮҸгҖӮиҜ·е…Ҳи®ҫзҪ®еҗҺеҶҚиҝҗиЎҢпјҲеӣҫзүҮе·Із”ҹжҲҗж—¶ж— йңҖйҮҚи·‘пјүгҖӮ"); process.exit(2); }

const ENDPOINT = "https://gpt-agent.cc/v1/images/generations";
const SIZE = "1024x1024"; // 1:1 designer-toy portrait

for (const d of [PORTRAIT_DIR]) if (!existsSync(d)) mkdirSync(d, { recursive: true });

async function genPortrait(slug, entry) {
  const file   = join(PORTRAIT_DIR, `${slug}.webp`);
  const file2x = join(PORTRAIT_DIR, `${slug}@2x.webp`);
  if (existsSync(file) && statSync(file).size > 8_000) {
    console.log(`SKIP  ${slug} (exists ${(statSync(file).size / 1024) | 0}KB)`);
    return { slug, ok: true, skipped: true };
  }
  const prompt = MASCOT_STYLE + entry.prompt + "." + RARITY_LOOK[entry.rarity];
  for (let attempt = 0; attempt < 3; attempt++) {
    try {
      const ctrl = new AbortController();
      const to = setTimeout(() => ctrl.abort(), 120_000);
      const r = await fetch(ENDPOINT, {
        method: "POST",
        headers: { "content-type": "application/json", authorization: "Bearer " + KEY },
        // background:"transparent" вҶ’ alpha cutout; harmless if the gateway ignores it
        body: JSON.stringify({ model: "gpt-image-2", prompt, n: 1, size: SIZE, background: "transparent" }),
        signal: ctrl.signal,
      });
      clearTimeout(to);
      if (!r.ok) { console.log(`  ${slug} attempt ${attempt + 1} HTTP ${r.status}: ${(await r.text()).slice(0, 120)}`); continue; }
      const j = await r.json();
      const b64 = j?.data?.[0]?.b64_json;
      const remote = j?.data?.[0]?.url;
      let buf = b64 ? Buffer.from(b64, "base64")
                    : remote ? Buffer.from(await (await fetch(remote)).arrayBuffer()) : null;
      if (!buf || buf.length < 20_000) { console.log(`  ${slug} attempt ${attempt + 1}: empty/too-small`); continue; }
      // keep alpha; trim transparent margin; two sizes. q72 lands ~40KB at 768ВІ.
      const base = sharp(buf).trim({ threshold: 10 });
      const w768  = await base.clone().resize({ width: 768,  withoutEnlargement: true }).webp({ quality: 72, alphaQuality: 90 }).toBuffer();
      const w1024 = await base.clone().resize({ width: 1024, withoutEnlargement: true }).webp({ quality: 74, alphaQuality: 90 }).toBuffer();
      writeFileSync(file, w768);
      writeFileSync(file2x, w1024);
      console.log(`OK    ${slug} -> ${(w768.length/1024)|0}KB / @2x ${(w1024.length/1024)|0}KB`);
      return { slug, ok: true, rarity: entry.rarity, hex: entry.hex };
    } catch (e) { console.log(`  ${slug} attempt ${attempt + 1} ERR ${e.message}`); }
  }
  console.log(`FAIL  ${slug}`);
  return { slug, ok: false };
}

async function pool(items, size, fn) {
  const out = []; let i = 0;
  await Promise.all(Array.from({ length: size }, async () => {
    while (i < items.length) { const idx = i++; out[idx] = await fn(items[idx]); }
  }));
  return out;
}

(async () => {
  const only = process.argv.slice(2);
  const slugs = Object.keys(MASCOTS).filter((s) => !only.length || only.includes(s));
  console.log(`# generating ${slugs.length} mascot portraits -> ${PORTRAIT_DIR}`);
  const res = await pool(slugs, 3, (s) => genPortrait(s, MASCOTS[s]));
  // emit manifest (UI reads slugвҶ’meta; never trust the FS at runtime)

  }));

    JSON.stringify({ generatedAt: new Date().toISOString(), mascots: manifest }, null, 2));
  const ok = res.filter((r) => r?.ok).length;
  const fail = res.filter((r) => r && !r.ok).map((r) => r.slug);
  console.log(`\n=== ${ok}/${slugs.length} ok${fail.length ? ", FAILED: " + fail.join(", ") : ""} ===`);
```

## 7. Fallback strategy (must never 404 / never block render)

Three independent layers, matching the existing `imageFailed`/`QuestCardFallbackArt` pattern:

1. **Per-image `onError`** вҶ’ swap `src` to `/mascots/fallback/mascot-fallback.webp` (a generated neutral claymation "?" toy on a `--ink-900` field). One generated asset covers all 14 slugs.
2. **Manifest gate** вҖ” the server component reads `public/manifest.mascots.json` and only passes a `slug` to the UI if it's present; an un-generated mascot degrades to the existing CSS-only `QuestBlindBoxArt` geometric frame (already in the component, no art needed). This is why the manifest is emitted by the pipeline, not derived at runtime from the filesystem.
3. **Frame fallback** вҖ” epic/legendary WebP foil missing вҶ’ component falls back to the `rare.svg` neon frame (still on-brand), logged once.

Because В§1 maps `slug === card.id`, the *existing* `quest-cards/front-${card.id}.png` is itself a tertiary fallback for the 12 core mascots during the migration window вҖ” set it as the `QuestCardArt` `onError` target before deleting the old PNGs.
